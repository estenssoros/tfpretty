# TF Pretty

make terraform output more pretty

## Gif of tests

![](https://github.com/estenssoros/tfpretty/blob/main/gif/render1696603062341.gif)

## How do

```
git clone https://github.com/estenssoros/tfpretty.git
cd tfpretty
go install
cd <your terraform code>
tfpretty apply
tfpretty destroy
```
